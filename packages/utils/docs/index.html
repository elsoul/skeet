Skeet Framework Plugin - Utils

Skeet Utils Plugin for Utitilies.

This plugin is a collection of utilities for basic operations.

  • Encryption
  • String
  • Number
  • Time
  • Http
  • Notification

Installation

$ skeet yarn add -p @skeet-framework/utils

or

$ yarn add @skeet-framework/utils

What's Skeet?

TypeScript Serverless Framework 'Skeet'.

The Skeet project was launched with the goal of reducing software development, operation, and maintenance costs.

Build Serverless Apps faster. Powered by TUtils, Cloud Functions, Typesaurus, Jest, Prettier, and Google Cloud.

Dependency

  • TypeScript
  • Node
  • Yarn
  • Google SDK
  • GitHub CLI
